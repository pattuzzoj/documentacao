content Define o valor de cada metadado ou http-equiv.
charset Define a codificação de caracteres do documento HTML.

http-equiv="content-language" Define o idioma padrão.
http-equiv="default-style" Define o nome para uma folha de estilo alternativa.
http-equiv="refresh" Define o tempo para atualização ou redirecionamento do documento HTML.
http-equiv="content-type" Define de forma alternativa a codificação de caracteres do documento HTML.
